# Keybinds

## Apps

| Keybind | Action |
|---------|--------|
| Option + Q | Open Kitty terminal |
| Option + C | Open Visual Studio Code |
| Option + S | Open Spotify |
| Cmd + B | Open Zen browser |
| Cmd + Return | Open Finder |

## Window Focus

| Keybind | Action |
|---------|--------|
| Option + Arrow / hjkl | Focus window in direction |

## Swap Windows

| Keybind | Action |
|---------|--------|
| Shift + Option + Arrow / hjkl | Swap window in direction |

## Resize

| Keybind | Action |
|---------|--------|
| Ctrl + Option + Left/Right / h/l | Shrink/expand width |
| Ctrl + Option + Up/Down / k/j | Shrink/expand height |

## Window State

| Keybind | Action |
|---------|--------|
| Ctrl + Z | Toggle float (centered) |
| Shift + Option + M | Toggle fullscreen |
| Option + O | Toggle split orientation |

## Switch Workspace

| Keybind | Action |
|---------|--------|
| Ctrl + h | Switch to prev workspace |
| Ctrl + l | Switch to next workspace |

## Move to Workspace

| Keybind | Action |
|---------|--------|
| Shift + Ctrl + Left / h | Move window to prev workspace |
| Shift + Ctrl + Right / l | Move window to next workspace |

## Layout

| Keybind | Action |
|---------|--------|
| Ctrl + Option + R | Rotate counter-clockwise |
| Ctrl + Option + Shift + R | Rotate clockwise |
| Ctrl + Option + B | Balance all windows |
| Ctrl + Option + 1 | BSP layout |
| Ctrl + Option + 2 | Stack layout |
| Ctrl + Option + 3 | Float layout |

## Other

| Keybind | Action |
|---------|--------|
| Ctrl + Option + Shift + 0 | Restart yabai |
